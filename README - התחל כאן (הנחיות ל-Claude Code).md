## 1. מה בונים (בקצרה)

ממשק (Wizard) ששואל מורה/רכז כמה שאלות ובונה אוטומטית **תוכנית עבודה שנתית** — לוח-שנה חודשי מ-1.9.26 עד 20.6.27, עם פריסת הנושאים, אירועים, משימות מודל והתראות על חריגות.

## 2. מקור-האמת — קְרא קודם

📌 **`מסמך אפיון v2 - ממשק תוכנית עבודה שנתית (ז-ח).md`** — זהו האפיון המחייב והעדכני. כל מה שבונים מתחיל ממנו. (קיים גם כ-`.docx` לקריאה נוחה.)

> מסמך ה-v1 הישן (`מסמך אפיון - ממשק תוכנית עבודה שנתית.md/.docx`) נשאר לרקע בלבד — במקרה של סתירה, **v2 גובר**.

## 3. היקף הגרסה הנוכחית

- בונים **רק לכיתות ז׳ ו-ח׳** (מובנות וכופות לפי תוכנית העבודה).
- **שכבת ט׳ לא נבנית עכשיו** — היא מתועדת בסעיף 11 באפיון (ענף גמישות: מדעים כללי מול דיסציפלינה בודדת).
- **בלי LLM** — מנוע חוקים דטרמיניסטי בלבד. וו ל-LLM יישאר לעתיד.
- פיצ׳רים שדורשים שרת (רישום/אימות, סטטוס מו״ט, גוגל קלנדר) — **בשלב הזה מדומים מקומית**; החיבור האמיתי בפאזה ב׳ (אפיון, סעיף 12–13).

## 5. סדר בנייה מומלץ (פאזה א׳)

1. **בנק נתונים** מובנה (JSON) מ-ז׳+ח׳ + הגאנט המחוזי.
2. **מסך רישום** (שם, אימייל, סמל מוסד) — שמירה מקומית.
3. **שאלון v2** (אפיון, סעיף 8) — כולל ביטול שאלות 2–3 ושינוי 4–5.
4. **מנוע חלוקת שעות** + מנגנון מחסור עם תיבות סימון (סעיף 8.2).
5. **שיבוץ משימות מודל + הכנה לחקר** (סעיף 9).
6. **פלט לוח-שנה חודשי** + טבלת תוכנית + ייצוא `.ics`/PDF/Word.
7. **התראות וחריגות** (סעיף 14).

## 6. ארכיטקטורה ומשאבים (לבחירת ה-stack)

מנוע דטרמיניסטי. למימוש האמיתי (Auth, DB, Google Calendar API) להיעזר בקטלוג Vibe Code:

- קטלוג כלים וספריות בעברית: https://vibe-code-best-practices.vercel.app/
- שרת MCP שמנתב כלי/ספרייה/Skill מהקטלוג (להתקנה ב-Claude Code: `claude mcp add`): https://github.com/arielaizn/vibe-code-mcp

Stack מוצע: Next.js + Tailwind + shadcn/ui + Drizzle/Postgres + Better Auth + Vercel, עם פונקציית שרת ל-Google Calendar API ו-LLM עתידי.

## 7. כללי עבודה

- **עברית ו-RTL מלא** בכל הממשק והתוצרים.
- דיוק מדעי — לא להמציא תכנים, שעות, תאריכים או קישורים.
- **מקורות רשמיים בלבד:** הנתונים נשאבים אך ורק מהמסמכים הרשמיים של משרד החינוך (מפרטי התכנים docx, הגאנט המחוזי, מיפוי משימות המודל) — **לא מהאפיון** ולא ממסמך נגזר. בנקי הנתונים בקוד (`app/src/data/`) חולצו ישירות מה-docx הרשמיים (ז׳=109 ש׳, ח׳=138 ש׳).
- ממשק נקי וידידותי למורה לא-טכנולוגי.
- **מקפים ב-UI הגלוי למשתמשים: רק מקף מקלדת `-`** — בלי en-dash `–` או em-dash `—` בשום טקסט שהמורה/הרכז רואים (כולל פלטי PDF/קלנדר). כלל פרויקט מחייב.
